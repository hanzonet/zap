# hanzo-zap

ZAP (Zero-copy Agent Protocol) wire protocol for Hanzo products

- crates.io: [`hanzo-zap`](https://crates.io/crates/hanzo-zap) (0.6.75)
- License: MIT

## Install

```toml
[dependencies]
hanzo-zap = "0.6.75"
```

## Part of Hanzo Network

This is one of 36 focused crates that compose the Hanzo Network. Each
lives in its own repository under [`github.com/hanzonet`](https://github.com/hanzonet)
to keep concerns separated and ship cadences independent.
